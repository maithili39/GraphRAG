# GraphRAG Hackathon Project (Neo4j Migration)

Please refer to [Guide.md](Guide.md) for the complete instructions on how to set up, ingest data, run the API/Frontend, and evaluate the pipelines.

## Quick Summary
- **Database:** Neo4j (via Docker)
- **Pipelines:** LLM-Only, Basic RAG (FAISS), GraphRAG (Neo4j Hybrid Graph)
- **Frontend:** React + Recharts dashboard
- **Eval:** Groq LLM-as-a-judge + BERTScore
